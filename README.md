# EventEase - Event Management System (Part 1)

## Project Overview
EventEase is a web-based application built using ASP.NET Core MVC. It is designed to help users manage venues, schedule events, and handle bookings efficiently. This project was developed as part of the PRSE module at Varsity College.

## Features (Phase 1)
* **Venue Management:** Add, view, and list different event locations.
* **Event Scheduling:** Create events with specific dates and descriptions.
* **Booking System:** Capture customer details for event reservations.
* **Image Support:** Integration of placeholder URLs for venue and event visuals.

## Technologies Used
* **Framework:** ASP.NET Core 8.0 (MVC)
* **Language:** C#
* **Database:** Entity Framework Core with SQL Server
* **Frontend:** Bootstrap & Razor Pages

## Setup Instructions
1. Clone the repository to your local machine.
2. Open the solution file (.sln) in Visual Studio 2022.
3. Open the **Package Manager Console**.
4. Run `Update-Database` to generate the local SQL database.
5. Press **F5** or click the Green Play button to run the application.
